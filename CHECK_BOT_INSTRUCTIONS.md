# Инструкция по проверке работы deep link в Telegram боте

## Шаги для проверки:

### 1. Убедитесь, что бот запущен
Откройте новое окно командной строки и выполните:
```
cd C:\SnowWhiteAI\GrantService
python telegram-bot\main.py
```

Если бот уже запущен, сначала остановите его (Ctrl+C), затем запустите заново.

### 2. Проверьте работу бота
1. Откройте Telegram
2. Найдите бота @GrantServiceHelperBot
3. Отправьте команду `/start`
4. Бот должен ответить главным меню

### 3. Проверьте команду /get_access
Отправьте боту команду:
```
/get_access
```
Бот должен сгенерировать токен доступа.

### 4. Проверьте deep link
Перейдите по ссылке в браузере или Telegram:
```
https://t.me/GrantServiceHelperBot?start=get_access
```

После перехода:
1. Нажмите кнопку START в боте
2. Бот должен автоматически сгенерировать токен

## Если deep link не работает:

### Вариант 1: Использовать прямую команду
Вместо deep link используйте команду напрямую:
```
/get_access
```

### Вариант 2: Проверить, что бот видит команды
1. Отправьте боту `/start`
2. Если бот отвечает - он работает
3. Отправьте `/get_access` для получения токена

## Альтернативный способ получения токена:

Если deep link не работает, но бот отвечает на команды:

1. Откройте @GrantServiceHelperBot
2. Отправьте команду `/get_access`
3. Скопируйте полученную ссылку с токеном
4. Используйте её для входа в админ-панель

## Проверка логов

Чтобы увидеть, что происходит с ботом:
```
type C:\SnowWhiteAI\GrantService\logs\telegram_bot.log
```
