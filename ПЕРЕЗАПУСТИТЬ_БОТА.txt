ВАЖНО: Чтобы deep link заработал, нужно перезапустить бота!

1. ОСТАНОВИТЕ ТЕКУЩИЙ БОТ:
   - Найдите окно где запущен бот (там где видно логи)
   - Нажмите Ctrl+C чтобы остановить

2. ЗАПУСТИТЕ БОТ ЗАНОВО:
   Дважды кликните на файл:
   C:\SnowWhiteAI\GrantService\restart_bot.bat
   
   ИЛИ выполните в командной строке:
   cd C:\SnowWhiteAI\GrantService
   python telegram-bot\main.py

3. ПРОВЕРЬТЕ РАБОТУ:
   После перезапуска:
   - Откройте @GrantServiceHelperBot в Telegram
   - Отправьте команду /get_access
   - Должен сгенерироваться токен

4. ПРОВЕРЬТЕ DEEP LINK:
   Перейдите по ссылке:
   https://t.me/GrantServiceHelperBot?start=get_access
   
   После нажатия START бот должен автоматически
   сгенерировать токен доступа.

ЕСЛИ НЕ РАБОТАЕТ DEEP LINK:
Используйте команду /get_access напрямую в боте - она точно работает!
